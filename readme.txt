Master branch에는 현재까지 완성된 부분만 올라가면 됩니다.
하위 노드에서 완성본이라고 판단되는 파일을 Master branch에 올리도록 요청하고, 올리시면 올라갑니다.
branch는 Master v 라고 된 부분을 클릭하시면, 
find or create라고 editText 공간이 나오는데, 거기에다가 자기가 치고 싶은 이름이나, 만들어야 하는 부분에 관한 정보를 담아 
(i.e. Schedule이면, Schedule이라고 치면 밑에 create branch: Schedule from master 라고 뜹니다.)
그거 클릭하시면 해당 branch가 Master node에서 분리되어 Master노드에 포함되어 있는 파일들과 함께, 해당 노드의 이름의 자식 노드로, 만들어집니다.
