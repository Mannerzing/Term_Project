기존 맵이 아무리 고쳐도 제 환경에서는 구글 본사 위치에 픽스되어서 카메라 이동, 검색, 자기 위치로 이동 모두 다 구동이 안됐습니다.
그래서, 일단은 뭐라도 구동을 해야하기에, 별도로 만든 버전으로 좌표값 따서, 리스트에 집어넣는 것으로 처리해버렸습니다.
(좌표값은 하나만 나오도록, 새로 찾거나, 지도상에 마킹을 할 경우 이전의 마커는 삭제하도록 해놨습니다.)
동적으로 리스트 뷰에 집어넣도록 코드를 만들었지만, 구동해보면 곧바로 알게 될 겁니다만.. 어딘가에 저장하는 것이 아니고,
매번 새로 불러오는 코드이기 때문에 temp로 들어갈때만 해당 값이 들어간 모습이 나올 겁니다. (이 부분은 따로 어딘가에 저장 시켜야 할 것 같은데, 일단은 구동에 초점을 맞췄기에 나중에 고쳐야 됩니다..)

분리 버전으로 만들었기 때문에, 합본으로 합칠 경우, 합치는 본체쪽에서 새로 연결되는 부분들에 대해서 연결을 해주셔야 됩니다.
